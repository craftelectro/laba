Инструкция по запуску
Установите Qt:
Скачайте и установите Qt Creator.
Создайте проект:
Откройте Qt Creator.
Выберите File > New File or Project > Qt Widgets Application.
Назовите проект, выберите компилятор (например, MinGW для Windows).
Скопируйте код:
Замените содержимое файла main.cpp кодом выше.
Скомпилируйте и запустите:
Нажмите Run (зелёная кнопка).
