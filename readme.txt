Инструкция по запуску
Установите Qt:
Скачайте и установите Qt Creator.
Создайте проект:
Откройте Qt Creator.
Выберите File > New File or Project > Qt Widgets Application.
Назовите проект, выберите компилятор (например, MinGW для Windows).
Скопируйте код:
Замените содержимое файла main.cpp кодом выше.
Скомпилируйте и запустите:
Нажмите Run (зелёная кнопка).


Введите данные процессов в текстовое поле, например:
0 8 0
1 4 1
2 9 2
Где:
0 — ID процесса,
8 — burst (время выполнения),
0 — время появления.
Выберите алгоритм из выпадающего списка (например, FCFS).
Нажмите Запустить — результаты отобразятся в таблице.
